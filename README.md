# XHCI-unsupported
**A fix driver for USB by RehabMan.**

## Supported id

`0x8cb1` for 9-series

`0x8d31` for X99

`0xa2af` for 200-series and X299

`0x9ded` `0xa36d` `0x02ed` for 300-series

`0xa3af` for 400-series

`0x43ed` `0xa0ed` for 500-series

`0x7ae0` for 600-series

`0x7a60` for 700-series
